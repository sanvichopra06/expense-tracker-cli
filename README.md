# Expense Tracker CLI

A beginner-friendly Python command-line application to track and manage daily expenses.

## Features
- Add expenses
- View all expenses
- Calculate total spending
- Find highest expense
- Find lowest expense
- Delete expenses

## Concepts Used
- Python lists
- Functions
- Loops
- Conditionals
- Traversal logic

## Tech Stack
- Python

## Future Improvements
- CSV file storage
- Expense categories
- Monthly summaries
- Data visualization
- Pandas integration
